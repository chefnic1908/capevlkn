# Cape V of LKN — Premium Website

## Uploading to GitHub Pages from your phone

1. Open the Cape V GitHub repository.
2. Upload or replace these three files:
   - `index.html`
   - `style.css`
   - `script.js`
3. Keep your existing `Logo.PNG`, `favicon.png`, and `CNAME` files in the repository.
4. Choose **Commit changes**.
5. Wait 1–3 minutes, then refresh `capevlkn.com`.

## Important filename note

The website currently references the exact filename `Logo.PNG`.
Do not rename it unless you also replace every `Logo.PNG` reference in `index.html`.

## Quote form

The form safely opens the visitor's email app with all event details addressed to:
`capev.lkn@gmail.com`

This requires no paid form service and works on GitHub Pages.

## Gallery photos

The first version uses high-quality sample food images hosted by Unsplash.
They can later be replaced by your own Cape V photographs. Upload your images to the repository
and change each image URL in the Gallery section of `index.html`.
