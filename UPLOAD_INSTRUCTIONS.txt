CAPE V ONE-FILE FIX

This version contains the design, scripts, logo and every photograph inside ONE index.html file.

GitHub upload:
1. Open your website repository.
2. Delete or replace the current index.html.
3. Upload this new index.html directly into the repository root—not inside another folder.
4. Commit the change.
5. Wait 1–3 minutes and refresh the live site.

You do not need to upload an assets folder, styles.css or script.js with this version.
